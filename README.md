# 📦 Sistema de Controle de Estoque

Sistema de terminal desenvolvido em Python para gerenciar o estoque de produtos, com menu interativo e operações completas de cadastro.

## 🚀 Funcionalidades

- ✅ Adicionar produto (com validação de duplicados)
- ✅ Listar todos os produtos cadastrados
- ✅ Atualizar quantidade de um produto
- ✅ Remover produto
- ✅  Persistência de dados em arquivo CSV 

## 🛠️ Tecnologias

- Python 3
- Dicionários para armazenamento de dados
- Módulo `csv` nativo (em implementação)

## 📖 Como executar

1. Clone este repositório:
```bash
git clone https://github.com/Arthur-Hagel/sistema-estoque.git
```

2. Entre na pasta do projeto:
```bash
cd sistema-estoque
```

3. Execute o sistema:
```bash
python estoque.py
```

## 💡 Sobre o projeto

Este projeto faz parte da minha jornada de aprendizado em Python, aplicando conceitos como estruturas de dados (dicionários), funções, loops, tratamento de condições e, em breve, manipulação de arquivos CSV para persistência de dados.
